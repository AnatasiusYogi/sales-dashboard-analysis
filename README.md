# Analisis Data Penjualan & Dashboard Interaktif

##  Deskripsi Proyek
Proyek ini bertujuan untuk menganalisis performa penjualan perusahaan berdasarkan dataset transaksi.  
Analisis dilakukan untuk mengidentifikasi pertumbuhan penjualan, tingkat profitabilitas, serta peluang peningkatan efisiensi bisnis melalui visualisasi dashboard interaktif.

---

##  Tujuan Analisis
- Menganalisis total penjualan dan total keuntungan
- Mengukur tingkat profitabilitas (Profit Margin)
- Mengidentifikasi produk dan kategori paling menguntungkan
- Menganalisis performa berdasarkan region dan segment pelanggan
- Membuat dashboard interaktif untuk mendukung pengambilan keputusan

---

##  Key Performance Indicators (KPI)

- **Total Sales** ‚Üí Total pendapatan yang dihasilkan  
- **Total Profit** ‚Üí Total keuntungan bersih  
- **Profit Margin** ‚Üí (Total Profit / Total Sales) √ó 100%  
- **Total Orders** ‚Üí Jumlah transaksi unik (COUNT DISTINCT Order_ID)  

---

##  Tools & Teknologi yang Digunakan

- **Microsoft Excel** ‚Üí Data cleaning & eksplorasi awal  
- **SQL (BigQuery)** ‚Üí Analisis dan pengolahan data  
- **Looker Studio** ‚Üí Visualisasi dashboard  
- **Git & GitHub** ‚Üí Version control dan dokumentasi proyek
  
---

##  Insight Utama

- Region dengan penjualan tertinggi belum tentu memiliki profit tertinggi.
- Kategori dengan margin terbesar menjadi kontributor utama keuntungan.
- Beberapa kategori memiliki margin rendah sehingga perlu evaluasi strategi harga atau biaya.
- Terdapat pola peningkatan penjualan pada periode tertentu (indikasi seasonal trend).
